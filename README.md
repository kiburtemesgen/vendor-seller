# vendor-seller

## Installation
$ git clone https://github.com/kiburtemesgen/vendor-seller.git
$ cd vendor-seller  #add .env file and insert MONGODB_URL, DATABASE_PASSWORD and JWT_SECRET fields 
$ npm install

## Running the app
$ npm run server
